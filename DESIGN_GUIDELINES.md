# Minimalist Design Guidelines (Medium-Inspired)

This document outlines the design principles for creating a clean, professional, and premium minimalist interface, similar to Medium, FollowDesign, and modern editorial sites.

## 1. Typography Hierarchy
*   **Body Text**: Use a high-quality Serif font (e.g., *Source Serif Pro*, *Charter*, or *Inter* for a modern touch) for long-form content.
*   **Headings**: Use a bold, high-contrast Sans-Serif or Serif. Keep heading levels distinct but not overwhelming.
*   **Line Height**: Maintain a generous line height (1.6 to 1.8) for readability.
*   **Tracking**: Avoid tight letter spacing. Let the characters breathe.

## 2. Color Palette
*   **Background**: Pure white (`#FFFFFF`) or a very soft off-white (`#FAFAFA`).
*   **Primary Text**: Deep charcoal (`#242424`) rather than pure black to reduce eye strain.
*   **Secondary Text**: Mid-gray (`#6B6B6B`) for metadata, dates, and less important info.
*   **Accents**: Use a single accent color sparingly (e.g., Medium green `#1A8917` or a sleek blue) for primary actions.

## 3. Layout & Whitespace
*   **Single Column focus**: For articles and content, use a centered single column (max-width: 680px - 720px).
*   **Generous Padding**: "White space is not empty space; it's a structural element." Use large gaps between sections.
*   **Grid System**: Use a simple, clean grid for list views with consistent gutters.

## 4. UI Elements
*   **Buttons**: Flat or very subtle borders. No heavy shadows or gradients. 
*   **Cards**: Minimalist cards with thin borders (`1px solid #E6E6E6`) or no borders at all, using whitespace to define boundaries.
*   **Icons**: Thin-stroke, consistent icons (e.g., Lucide or Phosphor Icons).

## 5. Interactions & Motion
*   **Hover Effects**: Subtle transitions (e.g., opacity changes or slight color shifts). No jarring movements.
*   **Loading States**: Smooth skeleton screens that maintain the layout structure.
*   **Micro-animations**: Use CSS transitions for link underlines or button states.

## 6. Content First
*   **No Placeholders**: Use real-looking dummy data.
*   **Imagery**: High-quality, purposeful images. Use rounded corners (4px - 8px) for a modern feel.
*   **Clutter-Free**: Remove unnecessary sidebars, popups, or decorative elements that don't serve the content.
